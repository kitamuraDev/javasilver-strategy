# 内容を訂正する

当サイトを閲覧している中で、文章やソースコードの間違いに気づいた場合は、以下の URL から Github の issue ページへ遷移することができます。
誤字脱字の修正から技術的な間違いまで、どのような報告でも感謝いたします。<br />
また、ソースコード（プログラム）の挙動、文章の意味や意図がわからないといった疑問を持つこともあるかと思います。そのような疑問もぜひ報告してください。

<br />

- [プログラムの挙動、文章の意味や意図などの質問をする](https://github.com/kitamuraDev/javasilver-strategy/issues/new?assignees=&labels=&template=question.md)
- [文章やソースコードの間違いを報告する](https://github.com/kitamuraDev/javasilver-strategy/issues/new?assignees=&labels=&template=report.md)
- [その他の Issue を立てたい場合はこちら](https://github.com/kitamuraDev/javasilver-strategy/issues/new?assignees=&labels=&template=%E3%81%9D%E3%81%AE%E4%BB%96.md)
